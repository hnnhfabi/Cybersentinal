# Cybersentinal
First project uploaded in GitHub. Done using Python. CyberSentinel is a Python + PyQt6 based interactive cybersecurity game.You play as a Junior Security Analyst defending Aethel Corp from a cyber attack to protect the Nexus engine.

Your mission is to protect the Nexus Engine by completing 4 cybersecurity tasks:

Mission 1 — Improve weak passwords and change them to stronger versions
Mission 2 — Decrypt intercepted enemy messages using Caesar/ROT13 cipher.
Mission 3 — Analyze login logs and identify the intruder’s suspicious IP.
Mission 4 —
Phase 1:Match the error to respective fix
Phase 2: Create  strong encrypted password to secure and close Nexus.

Complete all tasks to prevent the breach and become a Level IV Cyber Sentinel.Requires Python 3.10+ and PyQt6:
python pg1v2.py

This project gamifies learning the basics of:
-Password security
-Cryptography
-Log forensics
-Network firewalls
